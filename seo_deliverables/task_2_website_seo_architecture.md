# Task 2: Website SEO Architecture

This document outlines the recommended website SEO architecture for commonapostille.com to improve search engine visibility and user experience.

## Recommended URL Structure and Page List

A clean and logical URL structure is crucial for both users and search engines. The following structure is recommended for CNA's website:

*   **Homepage:** `commonapostille.com`
*   **Service Pages:** `commonapostille.com/service/[service-name]`
*   **Location Pages:** `commonapostille.com/locations/[city-name]`
*   **Country-Specific Apostille Pages:** `commonapostille.com/apostille/[country-name]`
*   **FAQ Page:** `commonapostille.com/faq`
*   **Blog:** `commonapostille.com/blog`
*   **Blog Posts:** `commonapostille.com/blog/[post-name]`
